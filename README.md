# 🚀 Space Escape Runner

A beginner-friendly mobile game built with React Native and Expo.

## 🎮 Features
- Custom spaceship built using React Native Views
- Left and right movement controls
- Falling asteroids
- Collision detection
- Score tracking
- High score storage using AsyncStorage
- Game Over and Restart functionality

## 🛠️ Tech Stack
- React Native
- Expo
- JavaScript
- AsyncStorage

## 📱 Testing
The app was tested on a real device using Expo Go.

## 📚 Learning Outcomes
This project helped me understand:
- React Native components
- State management with Hooks
- Game loops and animations
- Collision detection
- Mobile app development workflow
- Testing apps on a real device using Expo Go

## 🚀 Getting Started

### Clone the repository
```bash
git clone https://github.com/Niyatijain200511/SpaceEscapeRunner.git
```

### Install dependencies
```bash
npm install
```

### Start the Expo server
```bash
npx expo start
```

### Open on your phone
Install Expo Go and scan the QR code.
